ROLE DEFINITIONS

Admin

Full access to all system features and settings. Can manage users, teams, and permissions.

Key Permissions:

  • Manage users and permissions
  • Configure system settings
  • Access all deals and properties
  • Create and edit all content
  • Export and delete data
Analyst

Can analyze properties and deals, with limited management capabilities. Primary role for day-to-day work.

Key Permissions:

  • Create and edit assigned deals
  • View all deals and properties
  • Run analyses and generate reports
  • Upload and process documents
  • Export data (no deletion rights)
Viewer

Read-only access to approved content. Ideal for stakeholders who need visibility but not edit capabilities.

Key Permissions:

  • View assigned deals and properties
  • Access reports and dashboards
  • View document library (no upload)
  • Export permitted reports
  • No creation or edit rights
